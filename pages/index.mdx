{<h1 className="text-center font-extrabold md:text-5xl mt-8">AdaptUI React
Tailwind</h1>}

A React UI library built using collection of low level headless components/hooks
that are accessible, composable & customizable from
[Renderleskit React](https://github.com/timelessco/renderlesskit-react)
components & [tailwindcss](https://tailwindcss.com/).

## 🚀 Features

- Low level 🔥
- Accessible ♿
- Composable 🧰
- WAI-ARIA Compatible 🗒️
- React Hooks support ⚓
- Easy To Customize, Style & Extend 💅
- Themeable using tailwindcss 🎨
