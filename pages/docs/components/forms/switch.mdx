# Switch

Switch component is same as the Switch component but used separately & used to
toggle a single value.

<ComponentLinks github="switch" theme="switch" story="forms-switch" />

## Imports

```js
import {
  Switch,
  SwitchDescription,
  SwitchIcon,
  SwitchInput,
  SwitchLabel,
  useSwitchProps,
  useSwitchState,
  SwitchText,
} from "@renderlesskit/react-tailwind";
```

<Nextra.Callout>
  A complex component that supports customization as per the
  [composition](/docs/composition) guide.
</Nextra.Callout>

## Usage

<InteractiveCodeblock
  children={({ spreadProps }) => `<Switch ${spreadProps}>Label text</Switch>`}
  booleanProps={["disabled"]}
  themeProps={{ size: "switch.icon.size" }}
/>

## Switch sizes

Sizes can be set using the `size` prop. The default size is `md`. The available
sizes are: `sm` `md` `lg`

<Codeblock
  live
  code={`
<div className="inline-flex flex-col space-y-2">
  <Switch size="sm" defaultState={false} />
  <Switch size="md" defaultState={false} />
  <Switch size="lg" defaultState={false} />
</div>
`}
/>

## Switch Uncontrolled

Uncontrolled switch can be used with `defaultState` prop.

<Codeblock
  live
  code={`
<div className="inline-flex flex-col space-y-2">
  <Switch size="md" defaultState={true} />
  <Switch size="md" defaultState={false} />
</div>
`}
/>
