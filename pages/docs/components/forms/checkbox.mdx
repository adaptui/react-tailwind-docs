# Checkbox

The Checkbox is used in forms when a user needs to select multiple values from
several options. Checkbox composes Ariakit's
[checkbox](https://reakit.io/docs/checkbox/) component.

<ComponentLinks github="checkbox" theme="checkbox" story="forms-checkbox" />

## Imports

```js
import {
  Checkbox,
  CheckboxDescription,
  CheckboxIcon,
  CheckboxInput,
  CheckboxLabel,
  CheckboxText,
  CheckboxTextWrapper,
} from "@adaptui/react-tailwind";
```

</Nextra.Callout>

## Usage

<InteractiveCodeblock
  children={({ spreadProps }) =>
    `<Checkbox${spreadProps}>Label text</Checkbox>`
  }
  booleanProps={["disabled"]}
  themeProps={{ size: "checkbox.size" }}
  choiceProps={{ defaultValue: ["true", "false", "'indeterminate'"] }}
/>

## Checkbox sizes

Sizes can be set using the `size` prop. The default size is `md`. The available
sizes are: `sm` `md` `lg`

<Codeblock live noInline>

```
import { Checkbox } from "@adaptui/react-tailwind";

export const App = props => {
  return (
    <div className="inline-flex flex-col space-y-2">
      <Checkbox size="sm" defaultValue={false} />
      <Checkbox size="md" defaultValue={false} />
      <Checkbox size="lg" defaultValue={false} />
    </div>
  );
};
```

</Codeblock>

## Checkbox Uncontrolled

Uncontrolled checkbox can be used with `defaultState` prop.

<Codeblock live noInline>

```
import { Checkbox } from "@adaptui/react-tailwind";

export const App = props => {
  return (
    <div className="inline-flex flex-col space-y-2">
      <Checkbox size="md" defaultValue={true} />
      <Checkbox size="md" defaultValue={false} />
      <Checkbox size="md" defaultValue="indeterminate" />
    </div>
  );
};
```

</Codeblock>

## Checkbox UI states

<Codeblock live noInline>

```
import { Checkbox } from "@adaptui/react-tailwind";

export const App = props => {
  return (
    <div className="inline-flex w-72 flex-col space-y-2">
      <Checkbox size="md" defaultState={false} label="Checkbox" />
      <Checkbox
        size="md"
        defaultValue={false}
        label="Multilined label Checkbox, just for demonstration"
      />
      <Checkbox
        size="md"
        defaultValue={false}
        label="Checkbox"
        description="Used when the checkbox is selected and will use its value for the form submission."
      />
    </div>
  );
};
```

</Codeblock>
