# RadioGroup

RadiosGroup are used when only one choice may be selected in a series of
options. RadioGroup composes `Radio` component. It users Ariakit's
[Radio](https://reakit.io/docs/radio/) component internally.

<ComponentLinks github="radio-group" theme="radio" story="forms-radiogroup" />

## Imports

```js
import {
  RadioGroup,
  RadioGroupWrapper,
  RadioShowMore,
} from "@renderlesskit/react-tailwind";
```

## RadioGroup sizes

Sizes can be set using the `size` prop. The default size is `md`. The available
sizes are: `sm` `md` `lg`

<Codeblock live noInline>

```
import { RadioGroup, Radio } from "@adaptui/react-tailwind";

export const App = props => {
  const RadioGroupBasic = props => {
    return (
      <RadioGroup aria-label="fruits" {...props}>
        <Radio value="apple" label="Apple" />
        <Radio value="orange" label="Orange" />
        <Radio value="watermelon" label="Watermelon" />
        <Radio value="grapes" label="Grapes" />
      </RadioGroup>
    );
  };

  const RadioGroupStack = props => {
    return (
      <div className="flex space-x-2">
        <RadioGroupBasic size="sm" />
        <RadioGroupBasic size="md" />
        <RadioGroupBasic size="lg" />
      </div>
    );
  };

  return <RadioGroupStack />;
};
```

</Codeblock>

## RadioGroup Stack

RadioGroup can be arranged `vertical` or `horizontal`

<Codeblock live noInline>

```
import { RadioGroup, Radio } from "@adaptui/react-tailwind";

export const App = props => {
  const RadioGroupBasic = props => {
    return (
      <RadioGroup aria-label="fruits" {...props}>
        <Radio value="apple" label="Apple" />
        <Radio value="orange" label="Orange" />
        <Radio value="watermelon" label="Watermelon" />
        <Radio value="grapes" label="Grapes" />
      </RadioGroup>
    );
  };

  const RadioGroupStack = props => {
    return (
      <div className="flex flex-col space-y-2">
        <RadioGroupBasic size="md" stack="vertical" />
        <RadioGroupBasic size="md" stack="horizontal" />
      </div>
    );
  };

  return <RadioGroupStack />;
};
```

</Codeblock>

## RadioGroup Show More

RadioGroup with more Radios can be made hidden by adding `maxVisibleItems` prop

<Codeblock live noInline>

```
import { RadioGroup, Radio } from "@adaptui/react-tailwind";

export const App = props => {
  const RadioGroupBasic = props => {
    return (
      <RadioGroup aria-label="fruits" maxVisibleItems={2} {...props}>
        <Radio value="apple" label="Apple" />
        <Radio value="orange" label="Orange" />
        <Radio value="watermelon" label="Watermelon" />
        <Radio value="grapes" label="Grapes" />
      </RadioGroup>
    );
  };

  const RadioGroupStack = props => {
    return (
      <div className="flex flex-col space-y-2">
        <RadioGroupBasic size="md" stack="vertical" className="w-32" />
        <RadioGroupBasic size="md" stack="horizontal" />
      </div>
    );
  };

  return <RadioGroupStack />;
};
```

</Codeblock>
