# Input

The Input component is a text field used to get input from the user.

<ComponentLinks github="input" theme="input" story="forms-input" />

## Imports

```js
import {
  Input,
  InputBase,
  InputPrefix,
  InputSuffix,
  useInputProps,
  useInputState,
  InputWrapper,
} from "@renderlesskit/react-tailwind";
```

<Callout>
  A complex component that supports customization as per the
  [composition](/docs/composition) guide.
</Callout>

## Usage

<InteractiveCodeblock
  children={({ spreadProps }) => `<Input placeholder="Search" ${spreadProps}/>`}
  booleanProps={["disabled", "invalid", "loading"]}
  themeProps={{ variant: "input.base.variant", size: "input.base.size" }}
/>

## Input Variants

Input comes with 4 variants `outline` `subtle` `underline` & `ghost`.

<Codeblock
  live
  children={`
<div className="inline-flex flex-col space-y-2">
  <Input placeholder="Search" size="md" variant="outline" />
  <Input placeholder="Search" size="md" variant="subtle" />
  <Input placeholder="Search" size="md" variant="underline" />
  <Input placeholder="Search" size="md" variant="ghost" />
</div>
`}
/>

## Input sizes

Input comes with 4 sizes `sm` `md` `lg` & `xl`.

<Codeblock
  live
  children={`
<div className="inline-flex flex-col space-y-2">
  <Input placeholder="Search" size="sm" />
  <Input placeholder="Search" size="md" />
  <Input placeholder="Search" size="lg" />
  <Input placeholder="Search" size="xl" />
</div>
`}
/>

## Input Prefix/Suffix

You can append or prepend any element inside input. By default we don't give
spacing to the elements so you have to add spacing as needed.

Prefix is positioned absolutely within the input.

<Codeblock
  live
  children={`
<div className="inline-flex flex-col space-y-2">
  <Input placeholder="Search" prefix={<SlotIcon />} />
  <Input placeholder="Search" suffix={<SlotIcon />} />
  <Input placeholder="Search" prefix={<SlotIcon />} suffix={<SlotIcon />} />
</div>
`}
/>
