# Slider

The Slider is used to allow users to make selections from a range of values.

<ComponentLinks github="slider" theme="slider" story="forms-slider" />

## Imports

```js
import {
  Slider,
  SliderFilledTrack,
  SliderThumb,
  SliderThumbContainer,
  SliderThumbInput,
  SliderThumbWrapper,
  SliderTrack,
  SliderTrackWrapper,
  SliderWrapper,
} from "@adaptui/react-tailwind";
```

<Nextra.Callout>
  A complex component that supports customization as per the

</Nextra.Callout>

## Usage

<InteractiveCodeblock
  children={({ spreadProps }) =>
    `<Slider defaultValue={[50]} ${spreadProps} />`
  }
  booleanProps={["tooltip", "isDisabled"]}
  themeProps={{ size: "slider.size" }}

## Slider sizes

Sizes can be set using the `size` prop. The default size is `md`. The available
sizes are: `sm` `md` `lg` & `xl`.

<Codeblock live noInline>

```
import { Slider } from "@adaptui/react-tailwind";

export const App = props => {
  return (
    <div className="inline-flex flex-col space-y-2">
      <Slider size="sm" defaultValue={[50]} />
      <Slider size="md" defaultValue={[50]} />
      <Slider size="lg" defaultValue={[50]} />
      <Slider size="xl" defaultValue={[50]} />
    </div>
  );
};

```

</Codeblock>

## Slider KnobIcon

The knob icon can be customized by passing a custom icon for the SliderThumb.

<Codeblock live noInline>

```
import { Slider } from "@adaptui/react-tailwind";

export const App = props => {
  return (
    <div className="inline-flex flex-col space-y-2">
      <Slider size="md" defaultValue={[50]} knobIcon={SliderDefaultKnobIcon} />
    </div>
  );
};
```

</Codeblock>

## Slider states

Slider can have `min` and `max` values, `step`, `isDisabled` & `formatOptions`
as well.

<Codeblock
  live
  code={`
<div className="inline-flex flex-col space-y-2">
  <Slider size="md" defaultValue={[50]} minValue={20} maxValue={80} />
  <Slider size="md" defaultValue={[50]} minValue={20} maxValue={80} step={10} />
  <Slider size="md" defaultValue={[50]} minValue={20} maxValue={80} step={10} isDisabled={true} />
  <Slider size="md" defaultValue={[50]} formatOptions={{
      style: "unit",
      unit: "celsius",
      unitDisplay: "narrow",
    }} />
</div>
`}
/>

## Range Slider

Single Slider can be converted to a range slider using `range` prop.

<Codeblock
  live
  code={`
<div className="inline-flex flex-col space-y-2">
  <Slider size="md" defaultValue={[33, 66]} range />
</div>
`}
/>
