# Select

The Select component with collection of predefined options is used to pick a
value.

<ComponentLinks github="select" theme="select" story="forms-select" />

## Imports

```js
import {
  Select,
  SelectBase,
  SelectPrefix,
  SelectSuffix,
  useSelectProps,
  useSelectState,
  SelectWrapper,
} from "@renderlesskit/react-tailwind";
```

<Callout>
  A complex component that supports customization as per the
  [composition](/docs/composition) guide.
</Callout>

## Usage

<InteractiveCodeblock
  children={({ spreadProps }) => `
<Select ${spreadProps}>
  <option value="">Select anyone</option>
  <option value="dog">Dog</option>
  <option value="cat">Cat</option>
  <option value="hamster">Hamster</option>
  <option value="parrot">Parrot</option>
  <option value="spider">Spider</option>
  <option value="goldfish">Goldfish</option>
</Select>`}
  booleanProps={["disabled", "invalid", "loading"]}
  themeProps={{ variant: "select.base.variant", size: "select.base.size" }}
/>

## Select Prefix

Select comes only with `prefix` component but you can change the default
`suffix` component aswell.

<Codeblock
  live
  children={`
<div className="inline-flex flex-col space-y-2">
  <Select prefix={<SlotIcon />}>
    <option value="">Select anyone</option>
    <option value="dog">Dog</option>
    <option value="cat">Cat</option>
    <option value="hamster">Hamster</option>
  </Select>
</div>
`}
/>
