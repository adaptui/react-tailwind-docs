# Select

The Select component with collection of predefined options is used to pick a
value.

<ComponentLinks github="select" theme="select" story="forms-select" />

## Imports

```js
import {
  Select,
  SelectBase,
  SelectPrefix,
  SelectSuffix,
  SelectWrapper,
} from "@adaptui/react-tailwind";
```

</Nextra.Callout>

## Select Variants

Variants can be set using the `variant` prop. The default variant is `outline`.
The available variants are: `outline` `subtle` `underline` & `ghost`

<Codeblock live noInline>

```
import { Select } from "@adaptui/react-tailwind";

export const App = props => {
  return (
    <div className="inline-flex flex-col space-y-2">
      <Select variant="outline">
        <option value="">Select anyone</option>
        <option value="dog">Dog</option>
        <option value="cat">Cat</option>
        <option value="hamster">Hamster</option>
      </Select>
      <Select variant="subtle">
        <option value="">Select anyone</option>
        <option value="dog">Dog</option>
        <option value="cat">Cat</option>
        <option value="hamster">Hamster</option>
      </Select>
      <Select variant="underline">
        <option value="">Select anyone</option>
        <option value="dog">Dog</option>
        <option value="cat">Cat</option>
        <option value="hamster">Hamster</option>
      </Select>
      <Select variant="ghost">
        <option value="">Select anyone</option>
        <option value="dog">Dog</option>
        <option value="cat">Cat</option>
        <option value="hamster">Hamster</option>
      </Select>
    </div>
  );
};
```

</Codeblock>

## Select sizes

Sizes can be set using the `size` prop. The default size is `md`. The available
sizes are: `sm` `md` `lg` `xl`

<Codeblock live noInline>

```
import { Select } from "@adaptui/react-tailwind";

export const App = props => {
  return (
    <div className="inline-flex flex-col space-y-2">
      <Select size="sm">
        <option value="">Select anyone</option>
        <option value="dog">Dog</option>
        <option value="cat">Cat</option>
        <option value="hamster">Hamster</option>
      </Select>
      <Select size="md">
        <option value="">Select anyone</option>
        <option value="dog">Dog</option>
        <option value="cat">Cat</option>
        <option value="hamster">Hamster</option>
      </Select>
      <Select size="lg">
        <option value="">Select anyone</option>
        <option value="dog">Dog</option>
        <option value="cat">Cat</option>
        <option value="hamster">Hamster</option>
      </Select>
      <Select size="xl">
        <option value="">Select anyone</option>
        <option value="dog">Dog</option>
        <option value="cat">Cat</option>
        <option value="hamster">Hamster</option>
      </Select>
    </div>
  );
};
```

</Codeblock>

## Select Prefix

Comes only with `prefix` prop to append any content inside the Select. Also you
can change the default `suffix` component aswell.

<Codeblock live noInline>

```
import { Select } from "@adaptui/react-tailwind";

export const App = props => {
  return (
    <div className="inline-flex flex-col space-y-2">
      <Select prefix={<SlotIcon />}>
        <option value="">Select anyone</option>
        <option value="dog">Dog</option>
        <option value="cat">Cat</option>
        <option value="hamster">Hamster</option>
      </Select>
    </div>
  );
};
```

</Codeblock>
