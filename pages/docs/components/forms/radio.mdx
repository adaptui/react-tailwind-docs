# Radio

Radio composes Ariakit's [radio](https://reakit.io/docs/radio/) component. A
Single Radio doesn't make sense without `RadioGroup`.

<ComponentLinks github="radio" theme="radio" story="forms-radio" />

## Imports

```js
import {
  Radio,
  RadioDescription,
  RadioIcon,
  RadioInput,
  RadioLabel,
  RadioText,
  RadioTextWrapper,
} from "@adaptui/react-tailwind";
```

<Nextra.Callout>
  A complex component that supports customization as per the
  [composition](/docs/composition) guide.
</Nextra.Callout>

## Usage

<InteractiveCodeblock
  children={({ spreadProps }) =>
    `<Radio value="apple" label="Apple" ${spreadProps} />`
  }
  booleanProps={["disabled"]}
  themeProps={{ size: "radio.size" }}

## Radio sizes

Sizes can be set using the `size` prop. The default size is `md`. The available
sizes are: `sm` `md` `lg`

<Codeblock live noInline>

```
import { Radio } from "@adaptui/react-tailwind";

export const App = props => {
  return (
    <div className="inline-flex flex-col space-y-2">
      <Radio value="apple" label="Apple" size="sm" />
      <Radio value="apple" label="Apple" size="md" />
      <Radio value="apple" label="Apple" size="lg" />
    </div>
  );
};
```

</Codeblock>

## Radio UI states

Radio can be controlled using `state` prop.

<Codeblock live noInline>

```
import { Radio } from "@adaptui/react-tailwind";

export const App = props => {
  return (
    <div className="inline-flex flex-col space-y-2">
      <Radio size="md" value="apple" label="Apple" />
      <Radio size="md" value="apple" label="Apple" disabled="true" />
    </div>
  );
};
```

</Codeblock>

## Radio Disabled states

<Codeblock live noInline>

```
import { Radio } from "@adaptui/react-tailwind";

export const App = props => {
  return (
    <div className="inline-flex flex-col space-y-2">
      <Radio size="md" value="apple" label="Apple" disabled />
      <Radio size="md" value="apple" label="Apple" state="apple" disabled />
    </div>
  );
};
```

</Codeblock>

## Radio Label & Description

      <Radio
        size="md"
        value="apple"

        value="apple"
        label="Apple"

      />
    </div>
  );

</Codeblock>
