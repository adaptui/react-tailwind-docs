# Textarea

The Textarea component is a text field used to get textarea from the user.

<ComponentLinks github="textarea" theme="textarea" story="forms-textarea" />

## Imports

```js
import {
  Textarea,
  TextareaBase,
  TextareaGhost,
  TextareaIcon,
  useTextareaProps,
  useTextareaState,
  TextareaWrapper,
} from "@renderlesskit/react-tailwind";
```

<Callout>
  A complex component that supports customization as per the
  [composition](/docs/composition) guide.
</Callout>

## Usage

<InteractiveCodeblock
  children={({ spreadProps }) =>
    `<Textarea placeholder="Search" aria-label="Textarea Story Default Example" rowsMin={3} col={30} ${spreadProps}/>`
  }
  booleanProps={["disabled", "invalid", "loading"]}
  themeProps={{ variant: "textarea.base.variant", size: "textarea.base.size" }}
/>

## Textarea Variants

Textarea comes with 3 variants `outline` `subtle` & `underline`.

<Codeblock
  live
  children={`
<div className="inline-flex flex-col space-y-2">
  <Textarea placeholder="Search" aria-label="Textarea Story Default Example" rowsMin={3} col={30} variant="outline" />
  <Textarea placeholder="Search" aria-label="Textarea Story Default Example" rowsMin={3} col={30} variant="subtle" />
  <Textarea placeholder="Search" aria-label="Textarea Story Default Example" rowsMin={3} col={30} variant="underline" />
</div>
`}
/>

## Textarea sizes

Textarea comes with 4 sizes `sm` `md` `lg` & `xl`.

<Codeblock
  live
  children={`
<div className="inline-flex flex-col space-y-2">
  <Textarea placeholder="Search" aria-label="Textarea Story Default Example" rowsMin={3} col={30} size="sm" />
  <Textarea placeholder="Search" aria-label="Textarea Story Default Example" rowsMin={3} col={30} size="md" />
  <Textarea placeholder="Search" aria-label="Textarea Story Default Example" rowsMin={3} col={30} size="lg" />
  <Textarea placeholder="Search" aria-label="Textarea Story Default Example" rowsMin={3} col={30} size="xl" />
</div>
`}
/>

## Textarea autoSize

By setting `autoSize` prop the textarea will increase it's size according to the
content.

<Codeblock
  live
  children={`
<div className="inline-flex flex-col space-y-2 w-72" >
  <Textarea placeholder="Search" autoSize={true} defaultValue="Velit voluptatem a veritatis nam ducimus ut corporis. Iure dolorem perspiciatis nihil quam laudantium ea magnam est voluptatem. Quam repellat pariatur quasi accusantium aut architecto ut est. Ab totam voluptate beatae et alias quod vero enim tempora. Corporis laboriosam velit eveniet eos ut repellat pariatur ipsam." />
</div>
`}
/>

## Textarea Resize

`resize` prop contarints the resize direction of the textarea, It can accept 4
values `both | vertical | horizontal | none`

<Codeblock
  live
  children={`
<div className="inline-flex flex-col space-y-2">
  <Textarea placeholder="None" aria-label="Textarea Story Default Example" rowsMin={3} col={30} resize="none" />
  <Textarea placeholder="Both" aria-label="Textarea Story Default Example" rowsMin={3} col={30} resize="both" />
  <Textarea placeholder="Horizontal" aria-label="Textarea Story Default Example" rowsMin={3} col={30} resize="horizontal" />
  <Textarea placeholder="Vertical" aria-label="Textarea Story Default Example" rowsMin={3} col={30} resize="vertical" />
</div>
`}
/>
