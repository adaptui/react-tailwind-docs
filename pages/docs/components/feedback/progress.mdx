# Progress

Progress is used to display the progress status for a task that takes a long
time or consists of several steps. It uses AdaptUI Core's
[Progress](https://github.com/adaptui/react/blob/main/docs/progress.md)
component internally.

<ComponentLinks github="progress" theme="progress" story="feedback-progress" />

## Imports

```js
import {
  Progress,
  ProgressBar,
  ProgressHint,
  ProgressLabel,
  ProgressTrack,
  ProgressWrapper,
} from "@adaptui/react-tailwind";
```

<Nextra.Callout>
  A complex component that supports customization as per the
  [composition](/docs/composition) guide.
</Nextra.Callout>

## Usage

<InteractiveCodeblock
  children={({ spreadProps }) => `<Progress value={40} ${spreadProps} />`}
  themeProps={{ size: "progress.size" }}
/>

## Progress sizes

Sizes can be set using the `size` prop. The default size is `md`. The available
sizes are: `sm` `md` `lg` `xl`

<Codeblock live noInline>

```
import { Progress } from "@adaptui/react-tailwind";

export const App = props => {
  return (
    <div className="flex flex-col space-y-2">
      <Progress size="sm" value={20} />
      <Progress size="md" value={30} />
      <Progress size="lg" value={40} />
      <Progress size="xl" value={50} />
    </div>
  );
};
```

</Codeblock>

## Min max values

Set the `min` and `max` values of the progress bar.

<Codeblock live noInline>

```
import { Progress } from "@adaptui/react-tailwind";

export const App = props => {
  return <Progress min={10} max={40} value={35} />;
};
```

</Codeblock>

## Indeterminate progress

By setting the value prop to `null`, progress state becomes indeterminate

<Codeblock live noInline>

```
import { Progress } from "@adaptui/react-tailwind";

export const App = props => {
  return <Progress value={null} />;
};
```

</Codeblock>

## Progress Label

Progress also comes with A11y label that can also be used to describe the
progress status

<Codeblock live noInline>

```
import { Progress } from "@adaptui/react-tailwind";

export const App = props => {
  return <Progress value={50} label="Loading..." />;
};
```

</Codeblock>

## Progress Hint

Progress comes with A11y hint that can be used with label to indicate the
progress status.

<Codeblock live noInline>

```
import { Progress } from "@adaptui/react-tailwind";

export const App = props => {
  return <Progress value={50} label="Loading..." hint={50} />;
};
```

</Codeblock>

## Customizing Progress

To customize the appearance of the Progress component we can simply pass down
children components (ProgressTrack, ProgressBar) for more control.

<Codeblock live noInline>

```
import { Progress } from "@adaptui/react-tailwind";

export const App = props => {
  return (
    <Progress value={40}>
      <ProgressTrack className="bg-red-300" />
      <ProgressBar className="bg-red-800" />
    </Progress>
  );
};
```

</Codeblock>
