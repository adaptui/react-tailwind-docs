# Spinner

Spinners provide a visual cue that an action is either processing, awaiting a
course of change or a result.

<ComponentLinks github="spinner" theme="spinner" story="feedback-spinner" />

## Imports

```js
import { Spinner } from "@renderlesskit/react-tailwind";
```

## Usage

```jsx live
<Spinner />
```

## Spinner sizes

Spinner comes with 5 sizes `xs` `sm` `md` `lg` & `xl`

```jsx live
<div className="flex items-center space-x-2">
  <Spinner size="xs" />
  <Spinner size="sm" />
  <Spinner size="md" />
  <Spinner size="lg" />
  <Spinner size="xl" />
</div>
```

## Show stroke

To show the subtle stroke around the spinner we can use the `stroke` which
accepts `visible` or `transparent` values

```jsx live
<Spinner size="lg" stroke="visible" />
```
