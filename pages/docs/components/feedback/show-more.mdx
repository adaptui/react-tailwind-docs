# ShowMore

ShowMore is used to display the progress status for a task that takes a long
time or consists of several steps. It uses Ariakit's
[Disclosure](https://reakit.io/docs/disclosure/) component internally.

<ComponentLinks github="show-more" story="feedback-showmore" />

## Imports

```js
import {
  ShowMore,
  ShowMoreButton,
  ShowMoreContent,
} from "@adaptui/react-tailwind";
```

<Nextra.Callout>
  A complex component that supports customization as per the
  [composition](/docs/composition) guide.
</Nextra.Callout>

## ShowMore Directions

ShowMore component comes with two directions `vertical` & `horizontal`

<Codeblock live noInline>

```
import { ShowMore } from "@adaptui/react-tailwind";

export const App = props => {
  return (
    <div className="h-80 overflow-y-scroll">
      <div className="flex flex-col items-start space-y-2">
        <div>Apple</div>
        <div>Orange</div>
        <div>Watermelon</div>
        <div>Grapes</div>
        <div>Banana</div>
        <div>Blueberry</div>
        <ShowMore direction="vertical" className="flex flex-col space-y-2">
          <div>Sapota</div>
          <div>Papaya</div>
          <div>Avocado</div>
          <div>Strawberry</div>
          <div>Cherry</div>
          <div>Fig</div>

};
```
