# Avatar

Avatar component is used to display user's profile picture or initials or icons
along with badges.

<ComponentLinks github="avatar" theme="avatar" story="primitives-avatar" />

## Imports

```js
import {
  Avatar,
  AvatarDefaultStatusIndicators,
  AvatarIcon,
  AvatarImage,
  AvatarInitials,
  AvatarStatusIndicator,
  AvatarWrapper,

```

## Avatar sizes

Sizes can be set using the `size` prop. The default size is `xl`. The available
sizes are: `xs` `sm` `md` `lg` `xl` `2xl` `3xl`

<Codeblock live noInline>

## Avatar fallbacks

If there is an error loading the src of the avatar, there are 2 fallbacks:

- If there's a name prop, we use it to generate the initials.
- If there's no name prop, we use a default avatar.

If both name and src are not provided you can pass a `icon` prop to show any
content, `icon` has the lowest priority thus if name or src is provided fallback
will be ignored.

<Codeblock live noInline>

```
import { Avatar } from "@adaptui/react-tailwind";

export const App = props => {
  return (
    <div className="flex space-x-2">
      <Avatar
        size="xl"
        name="Chelsey Dietrich"
        src="https://bit.ly/broken-link"
      />
      <Avatar
        size="xl"
        name="Dennis Schulist"
        src="https://bit.ly/broken-link"
      />
      <Avatar size="xl" src="https://bit.ly/broken-link" />
      <Avatar size="xl" icon={<InfoCircleIcon />} />
    </div>
  );
};
```

</Codeblock>

## Avatar with status indicators

You can show status icons at the right corner of the Avatar using `status` prop.
Each avatar sizes has its own status indicators sizes.

<Codeblock live noInline>

```

export const App = props => {
  return (
    <div className="flex space-x-2">
      <Avatar
        status="active"
        size="xl"
        name="Leanne Graham"
        src="https://i.pravatar.cc/300?img=1"
      />
      <Avatar
        status="sleep"
        size="xl"

      />
      <Avatar
        status="away"
        size="xl"
```
