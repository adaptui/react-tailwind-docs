# Avatar

`Avatar` component is used to display user's profile picture, initials or icons.

<ComponentLinks github="avatar" theme="avatar" story="primitives-avatar" />

## Imports

```js
import {
  Avatar,
  AvatarIcon,
  AvatarImage,
  AvatarInitials,
  AvatarStatusIndicator,
  AvatarWrapper,
  useAvatarProps,
  useAvatarState,
  AvatarDefaultStatusIndicators,
} from "@renderlesskit/react-tailwind";
```

## Usage

<InteractiveCodeblock
  children={({ spreadProps }) =>
    `<Avatar src="https://bit.ly/dan-abramov" name="Dan Abramov" circular={false} ${spreadProps} />`
  }
  booleanProps={["circular"]}
  themeProps={{ size: "avatar.icon.size", status: "avatar.statusIndicators" }}
/>

## Avatar sizes

Avatar has 5 sizes `xs` `sm` `md` `lg` `xl` `2xl` `3xl`

```jsx live
<div className="flex space-x-2">
  <Avatar size="xs" src="https://bit.ly/code-beast" name="Code Beast" />
  <Avatar size="sm" src="https://bit.ly/ryan-florence" name="Rayan Florence" />
  <Avatar size="md" src="https://bit.ly/kent-c-dodds" name="Kent Dodds" />
  <Avatar size="lg" src="https://bit.ly/dan-abramov" name="Dan Abramov" />
  <Avatar size="xl" src="https://bit.ly/31ycy7V" name="Timelessco" />
  <Avatar
    size="2xl"
    src="https://i.pravatar.cc/300??img=61"
    name="Clint Barton"
  />
  <Avatar size="3xl" src="https://i.pravatar.cc/300??img=42" name="Jhon Snow" />
</div>
```

## Avatar fallbacks

If there is an error loading the src of the avatar, there are 2 fallbacks:

- If there's a name prop, we use it to generate the initials.
- If there's no name prop, we use a default avatar.

If both name and src are not provided you can pass a `icon` prop to show any
content, `icon` has the lowest priority thus if name or src is provided fallback
will be ignored.

```jsx live
<div className="flex space-x-2">
  <Avatar size="xl" name="Navin Moorthy" src="https://bit.ly/broken-link" />
  <Avatar size="xl" name="Anurag Hazra" src="https://bit.ly/broken-link" />
  <Avatar size="xl" src="https://bit.ly/broken-link" />
  <Avatar size="xl" icon={<InfoCircleIcon />} />
</div>
```
