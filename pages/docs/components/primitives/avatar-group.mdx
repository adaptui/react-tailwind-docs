# AvatarGroup

AvatarGroup is a component that displays a group of avatars.

<ComponentLinks
  github="avatar-group"
  theme="avatar"
  story="primitives-avatargroup"
/>

## Imports

```js
import {
  AvatarGroup,
  useAvatarGroupProps,
  useAvatarGroupState,
  AvatarGroupWrapper,
} from "@renderlesskit/react-tailwind";
```

<Nextra.Callout>
  A complex component that supports customization as per the
  [composition](/docs/composition) guide.
</Nextra.Callout>

## Usage

<InteractiveCodeblock
  children={({ spreadProps }) => `
<AvatarGroup circular={false} showRing={false} ${spreadProps}>

  booleanProps={["circular", "showRing"]}
  themeProps={{ size: "avatar.icon.size" }}
/>

## AvatarGroup sizes

Sizes can be set using the `size` prop. The default size is `"xl"`. The
available sizes are: `xs` `sm` `md` `lg` `xl` `2xl` `3xl`

<Codeblock
  live
  noInline
  children={`

  return (
    <AvatarGroup {...props}>

      <Avatar name="Ervin Howell" src="https://i.pravatar.cc/300?img=11" />
      <Avatar name="Clementine Bauch" src="https://i.pravatar.cc/300?img=21" />
      <Avatar name="Patricia Lebsack" src="https://i.pravatar.cc/300?img=31" />

<Reakit.Group
  aria-label="group of avatar groups"

## AvatarGroup props

### `max`

Set `max` to limit the number of avatars in the group. By default AvatarGroup
will display all avatars.

<Codeblock
  live
  children={`
<AvatarGroup max={3}>
  <Avatar name="Leanne Graham" src="https://i.pravatar.cc/300?img=1" />
  <Avatar name="Ervin Howell" src="https://i.pravatar.cc/300?img=11" />
  <Avatar name="Clementine Bauch" src="https://i.pravatar.cc/300?img=21" />
  <Avatar name="Patricia Lebsack" src="https://i.pravatar.cc/300?img=31" />
  <Avatar name="Chelsey Dietrich" src="https://i.pravatar.cc/300?img=41" />
  <Avatar name="Dennis Schulist" src="https://i.pravatar.cc/300?img=51" />
</AvatarGroup>
`}
/>

### `ringColor`

Set `ringColor` to change the color of the ring around the avatars. By default
it is `"ring-white"`. This props can be used to match the parent's background to
the AvatarGroup.

<Codeblock
  live
  children={`
<div className="flex h-20 w-40 items-center justify-center bg-orange-200">
  <AvatarGroup max={3} ringColor="ring-orange-200">

    <Avatar name="Ervin Howell" src="https://i.pravatar.cc/300?img=11" />
    <Avatar name="Clementine Bauch" src="https://i.pravatar.cc/300?img=21" />
