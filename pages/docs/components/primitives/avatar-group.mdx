# AvatarGroup

`AvatarGroup` component is used to display a set of avatars closed together.

<ComponentLinks
  github="avatar-group"
  theme="avatar"
  story="primitives-avatargroup"
/>

## Imports

```js
import {
  AvatarGroup,
  useAvatarGroupProps,
  useAvatarGroupState,
  AvatarGroupWrapper,
} from "@renderlesskit/react-tailwind";
```

## Usage

<InteractiveCodeblock
  children={({ spreadProps }) => `
<AvatarGroup circular={false} ${spreadProps}>
  <Avatar name="Leanne Graham" src="https://i.pravatar.cc/300?img=1" />
  <Avatar name="Ervin Howell" src="https://i.pravatar.cc/300?img=11" />
  <Avatar name="Clementine Bauch" src="https://i.pravatar.cc/300?img=21" />
  <Avatar name="Patricia Lebsack" src="https://i.pravatar.cc/300?img=31" />
  <Avatar name="Chelsey Dietrich" src="https://i.pravatar.cc/300?img=41" />
  <Avatar name="Dennis Schulist" src="https://i.pravatar.cc/300?img=51" />
</AvatarGroup>
`}
  booleanProps={["circular"]}
  themeProps={{ size: "avatar.icon.size" }}
/>

## AvatarGroup sizes

AvatarGroup has all 7 avatar sizes `xs` `sm` `md` `lg` `xl` `2xl` `3xl`

<Codeblock
  live
  noInline
  children={`

  return (
    <AvatarGroup {...props}>

      <Avatar name="Ervin Howell" src="https://i.pravatar.cc/300?img=11" />
      <Avatar name="Clementine Bauch" src="https://i.pravatar.cc/300?img=21" />
      <Avatar name="Patricia Lebsack" src="https://i.pravatar.cc/300?img=31" />

const AvatarGroupStack = props => { return (
