# Badge

Badge is a component that displays an indicator that requires attention. It can
also be used to display a status

<ComponentLinks github="badge" themeColor="badge" story="primitives-badge" />

## Imports

```js
import { Badge } from "@adaptui/react-tailwind";
```

## Usage

<InteractiveCodeblock
  children={({ spreadProps }) => `<Badge${spreadProps}>Click me</Badge>`}
  themeProps={{
    size: "badge.size",
    variant: "badge.themeColor.base",
    themeColor: "badge.themeColor",
  }}
/>

## Badge variants

Variants can be set using the `variant` prop. The default variant is `solid`.
The available variants are: `solid` `subtle` `outline`

<Codeblock live noInline>

```
import { Badge } from "@adaptui/react-tailwind";

export const App = props => {
  return (
    <div className="space-x-2">
      <Badge variant="solid">Solid</Badge>
      <Badge variant="subtle">Subtle</Badge>
      <Badge variant="outline">Outline</Badge>
    </div>
  );
};
```

</Codeblock>

## Badge Sizes

Sizes can be set using the `size` prop. The default size is `md`. The available
sizes are: `sm` `md` `lg`

<Codeblock live noInline>

```
import { Badge } from "@adaptui/react-tailwind";

export const App = props => {
  return (
    <div className="space-x-2">
      <Badge size="sm">Small</Badge>
      <Badge size="md">Medium</Badge>
      <Badge size="lg">Large</Badge>
    </div>
  );
};
```

</Codeblock>

## Badge Themes

Theme can be set using the `theme` prop. The default theme is `default`. The
available themes are: `default` `primary` `secondary` `success` `danger`

<Codeblock live noInline>

```

export const App = props => {
  return (
    <div className="flex flex-col space-y-1">

          Beta
        </Badge>
        <Badge size="md" variant="subtle" themeColor="default">
          Beta
        </Badge>
        <Badge size="md" variant="outline" themeColor="default">
          Beta
        </Badge>
      </div>
      <div className="space-x-2">
        <Badge size="md" variant="solid" themeColor="primary">
          Beta
        </Badge>
        <Badge size="md" variant="subtle" themeColor="primary">
          Beta
        </Badge>
        <Badge size="md" variant="outline" themeColor="primary">
          Beta
        </Badge>
      </div>
      <div className="space-x-2">
        <Badge size="md" variant="solid" themeColor="secondary">
          Beta
        </Badge>
        <Badge size="md" variant="subtle" themeColor="secondary">
          Beta
        </Badge>
        <Badge size="md" variant="outline" themeColor="secondary">
```

## Badge prefix

You can pass prefix props to the badge to prepend any content inside of badge.

<Codeblock live noInline>

```
import { Badge } from "@adaptui/react-tailwind";

export const App = props => {
  return (
    <div className="space-x-2">
      <Badge prefix={<SlotIcon />}>Search</Badge>
    </div>
  );
};
```

</Codeblock>
