# ButtonGroup

The ButtonGroup component can be used to group collection of buttons of same
type to perform the a set of actions.

<ComponentLinks
  github="button-group"
  theme="buttonGroup"
  story="primitives-buttongroup"
/>

## Imports

```jsx
import { ButtonGroup } from "@renderlesskit/react-tailwind";
```

- **ButtonGroup:** Used to group related buttons together in a horizontal stack,
  can also be set as `collapsed`

## Usage

<InteractiveCodeblock
  children={({ spreadProps }) => `
<ButtonGroup ${spreadProps}>
  <Button>Undo</Button>
  <Button>Redo</Button>
  <Button>Copy</Button>
  <Button>Paste</Button>
  <Button>Delete</Button>
  <Button>Close</Button>
</ButtonGroup>
    `}
  booleanProps={["collapsed"]}
  themeProps={{ size: "button.size.common", variant: "button.variant.common" }}
/>

## Button prefix, suffix & iconOnly

You can pass iconOnly, prefix and suffix props to the button to append or
prepend any content inside of button.

<Codeblock
  live
  noInline
  children={`
const ButtonGroupBasic = props => {
  return (
    <ButtonGroup size="md" variant="solid">
      <Button {...props}>Undo</Button>
      <Button {...props}>Redo</Button>
      <Button {...props}>Copy</Button>
      <Button {...props}>Paste</Button>
      <Button {...props}>Delete</Button>
      <Button {...props}>Close</Button>
    </ButtonGroup>
  );
};

render(

<div className="space-y-2 flex flex-col">
  <ButtonGroupBasic iconOnly={<SlotIcon />} />
  <ButtonGroupBasic prefix={<SlotIcon />} />
  <ButtonGroupBasic suffix={<SlotIcon />} />
  <ButtonGroupBasic prefix={<SlotIcon />} suffix={<SlotIcon />} />
</div>
) `} />
