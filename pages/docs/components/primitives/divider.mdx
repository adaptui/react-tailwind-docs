# Divider

Divider is a static structure that provides a visible boundary for elements. It
uses reakit's [Separator](https://reakit.io/docs/separator/) component
internally.

<ComponentLinks github="divider" story="primitives-divider" />

## Imports

```js
import { Divider } from "@adaptui/react-tailwind";
```

## Usage

<InteractiveCodeblock
  children={({ spreadProps }) => `<Divider${spreadProps} />`}
/>

## Divider direction

Divider comes with two directions: `horizontal` and `vertical`.

<Codeblock live noInline>

```
import { Divider } from "@adaptui/react-tailwind";

export const App = props => {
  return (
    <div className="flex flex-col space-y-2">
      <div className="w-80">
        <Divider orientation="horizontal" />
      </div>
      <div className="flex h-80 items-center justify-center">
        <Divider orientation="vertical" />
      </div>
    </div>
  );
};
```

</Codeblock>
## Divider label

Divider can have a label at the center to mark it as a section.

<Codeblock live noInline>

```
import { Divider } from "@adaptui/react-tailwind";

export const App = props => {
  return (
    <div className="flex flex-col space-y-2">
      <Divider />
      <Divider label={<Button variant="outline">Continue</Button>} />
    </div>
  );
};
```

</Codeblock>
