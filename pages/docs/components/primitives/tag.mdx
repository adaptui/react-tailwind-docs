# Tag

Tag is used when items need to be labelled, categorized, and organized using
keywords that describe them.

<ComponentLinks github="tag" theme="tag" story="primitives-tag" />

## Imports

```js
import { Tag } from "@adaptui/react-tailwind";
```

- **Tag** Standalone tag component

## Tag sizes

Sizes can be set using the `size` prop. The default size is `md`. The available
sizes are: `sm` `md` `lg` `xl`

<Codeblock live noInline>

```
import { Tag } from "@adaptui/react-tailwind";

export const App = props => {
  return (
    <div className="space-x-2">
      <Tag size="sm">Small</Tag>
      <Tag size="md">Medium</Tag>
      <Tag size="lg">Large</Tag>
      <Tag size="xl">Extra Large</Tag>
    </div>
  );
};
```

</Codeblock>

## Tag variants

Variants can be set using the `variant` prop. The default variant is `solid`.
The available variants are: `solid` `subtle` `outline`

<Codeblock live noInline>

```
import { Tag } from "@adaptui/react-tailwind";

export const App = props => {
  return (
    <div className="space-x-2">
      <Tag variant="solid">Solid</Tag>
      <Tag variant="subtle">Subtle</Tag>
      <Tag variant="outline">Outline</Tag>
    </div>
  );
};
```

</Codeblock>

<Nextra.Callout>
  You can add extra variants & sizes via the theme file. Checkout [theming
  guide](/docs/theming).
</Nextra.Callout>

## Closable Tag

Tags can be closable or removable. If `closable` prop is passed tag will show a
close icon in the `suffix`. You can override the default icon inside `suffix`.
Tag is by default renders a button component so you can pass the clickable props
directly.

<Codeblock live noInline>

```
import { Tag } from "@adaptui/react-tailwind";

export const App = props => {
  return <Tag closable>Removable</Tag>;
};
```

</Codeblock>

## Tag Prefix & Suffix

Same as buttons Tag also accepts `prefix` & `suffix` props which can be used to
append or prepend content in the tag.

<Codeblock live noInline>

```
import { Tag } from "@adaptui/react-tailwind";

export const App = props => {
  return (
    <div className="space-x-2">
      <Tag prefix={<SlotIcon />}>Search</Tag>
      <Tag closable suffix={<SlotIcon />}>
        Search
      </Tag>
    </div>
  );
};
```

</Codeblock>

<Nextra.Callout>
  Suffix will only be rendered if `closable` is set to `true`
</Nextra.Callout>
