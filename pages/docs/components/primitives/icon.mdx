# Icon

Icon is the base svg primitive for creating accssible icons.

<ComponentLinks github="icon" story="primitives-icon" theme="icon" />

## Imports

```js
import { Icon, createIcon } from "@adaptui/react-tailwind";
```

## Usage

<InteractiveCodeblock
  children={({
    spreadProps,
  }) => `<Icon viewBox="0 0 200 200" className="text-red-500" ${spreadProps}>
  <path
    fill="currentColor"
    d="M 100, 100 m -75, 0 a 75,75 0 1,0 150,0 a 75,75 0 1,0 -150,0"
  />
</Icon>`}
/>

If no children is provided Icon will render a fallback icon.

<Codeblock live noInline>

```
import { Icon } from "@adaptui/react-tailwind";

export const App = (props) => {
  return <Icon className="text-gray-800" />;
};
```

</Codeblock>

## `createIcon` utility

To create icons more easily we provide a `createIcon` utility which takes few
props and returns a React component;

<Codeblock live noInline>

```
import { createIcon } from "@adaptui/react-tailwind";

export const App = props => {
  const CheckIcon = createIcon({
    displayName: "CheckIcon",
    viewBox: "0 0 12 12",
    path: (
      <path
        fill="none"
        stroke="currentColor"
        strokeLinecap="round"
        strokeLinejoin="round"
        strokeWidth="1.3"
        d="M2.25 6.214l2.41 2.679L9.75 3"
      />
    ),
  });

  return <CheckIcon className="text-gray-800" />;
};
```

</Codeblock>
