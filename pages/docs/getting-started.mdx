# Getting Started

AdaptUI uses tailwind to provide better styling at one place, so you'll need to
setup a tailwind project using thier
[installation](https://tailwindcss.com/docs/installation) guide.

## Installation

You can either install with NPM or Yarn

  <Nextra.Tab>
    ```bash
    npm i @adaptui/react-tailwind
    ```
  </Nextra.Tab>
  <Nextra.Tab>
    ```bash
    yarn add @adaptui/react-tailwind

    ```

  </Nextra.Tab>

</Nextra.Tabs>

## Tailwind setup

After setting up tailwind in your project, import our `preset()` function inside
your `tailwind.config.js` and use it.

```js
module.exports = {
  presets: [require("@adaptui/react-tailwind/preset")],
  content: [
    // Make sure to add this purge to generate the component's default styling
    "node_modules/@adaptui/react-tailwind/**/*",
  ],
});
```

## Setting up provider

AdaptUI needs a theme provider to pass down all the neccesary styling for the
components.

Go to the root of your application and add the below code. Root of your app
might be `App.js` or `App.tsx` or `index.js`

```js
import * as React from "react";
// 1. import `AdaptUIProvider` component
import { AdaptUIProvider } from "@adaptui/react-tailwind";

function App() {
  // 2. Wrap AdaptUIProvider at the root of your app
  return (
    <AdaptUIProvider>
      <App />
    </AdaptUIProvider>
  );
}
```
