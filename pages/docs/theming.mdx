# Customize theme

If you need to customize the default styling for any components, you can do so
by extending the `DefaultTheme`

Our `AdaptUIProvider` accepts a `theme` prop which can be used to provide a
custom theme of your choice.

```jsx filename="App.js" /theme/2-3
// Pass the full theme object to `AdaptUIProvider` from the theme file.
<AdaptUIProvider theme={theme}>
  <App />
</AdaptUIProvider>
```

## Extending theme tokens

When using AdaptUI, The theme can be extended even futher with `extend` prop
which can be added to `AdaptUIProvider`

To extend or override a token in the default theme simply add a new file
`adaptui.config.js` & export a theme object. Use a `extend` key to extend a
theme token or simply to overwrite a token in the default theme object, create a
new object of same name.

For example, if you want to add a new color variant to the button you can do
this:

<Nextra.Tabs items={['JS', 'TS']} defaultIndex={1}>
  <Nextra.Tab>

```js filename="adaptui.config.js"
export const theme = {
  extend: {
    button: {
      themeColor: {
        base: {
          tertiary: { default: "bg-purple-600 text-white-900" },
        },
      },
      size: {
        xxl: {
          base: "h-14 min-w-14 px-6 rounded-xl text-xl",
        },
      },
    },
  },
};
```

  </Nextra.Tab>
  <Nextra.Tab>

```tsx filename="adaptui.config.ts"
import { extendTheme } from "@adaptui/react-tailwind";

export const theme = extendTheme({
  extend: {
    button: {
      themeColor: {
        base: {
          tertiary: { default: "bg-purple-600 text-white-900" },
        },
      },
      size: {
        xxl: {
          base: "h-14 min-w-14 px-6 rounded-xl text-xl",
        },
      },
    },
  },
});
```

  </Nextra.Tab>
</Nextra.Tabs>

Now inside your `<AdaptUIProvider>` you can use extend prop to pass the theme.
If you haven't already set up the provider you can
[refer this guide](/docs/getting-started#setting-up-provider)

```jsx
// Pass the new theme to `AdaptUIProvider` from the config file.
<AdaptUIProvider extend={theme}>
  <App />
</AdaptUIProvider>;

// Now you can use these colors in your components
function Usage() {
  return <Button variant="tertiary">Welcome</Button>;
}
```

If you are wondering about which keys of theme you can extend you can look at
the source code for each specific
[theme file](https://github.com/adaptui/adaptui-react-tailwind/tree/main/src/theme/defaultTheme)

Or if you have setup typescript you can also get autocompletion of those theme
keys, checkout [TypeScript Guide](/guides/typescript-guide).
